About This Project

How This Project is Important

We understand the difficulties in job searching and want to provide an efficient tool for students to find job opportunities quickly.

Project Architecture

This web application is a demo of a Highly Available and Auto-Scaling Containerized Application Deployment in AWS. It utilizes AWS infrastructure components such as:

  • Amazon ECS or EKS for container orchestration
  • Elastic Load Balancer (ELB) for high availability
  • Amazon RDS PostgreSQL for reliable database hosting
  • Auto Scaling Groups to handle traffic spikes
  • Amazon S3 and CloudFront for static assets (optional)

The application is containerized using Docker and deployed using CI/CD pipelines to ensure rapid and stable updates.

Technologies Used

  • Node.js (Express) for backend APIs
  • PostgreSQL via Amazon RDS
  • USAJobs API for real-time job listings
  • Chart.js and SweetAlert2 for interactive frontend
  • Docker and AWS ECS/Fargate (or EKS) for deployment

Powered by AWS, USAJobs API, and PostgreSQL
